### Example

Delete data from personal space.

```json
{
     "action": "DeleteData",
     "settings": {
         "filename": "data.csv"
     }
}
```

Delete data from space with ID `25180576-755b-46e1-8683-12062584e52c`.

```json
{
     "action": "DeleteData",
     "settings": {
         "filename": "data.csv",
         "spaceid" : "25180576-755b-46e1-8683-12062584e52c"
     }
}
```
